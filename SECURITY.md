# Security Policy

This is a static personal website — there is no backend, no database, no user
accounts, and no data collection.

## Reporting a vulnerability

If you notice a security issue (for example, a misconfigured header or an
exposed file that shouldn't be public), please report it privately rather than
opening a public issue:

- Use GitHub's [private vulnerability reporting](https://github.com/medi-v/personal-website/security/advisories/new), or
- Send a message via [LinkedIn](https://www.linkedin.com/in/sergeyvoelker/).

I'll respond as soon as I can. Thank you for helping keep the site safe.
